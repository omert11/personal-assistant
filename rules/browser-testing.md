# Browser & Frontend Testing

## Tercih Sirasi
- Tarayici otomasyonu ve frontend testi icin **Playwright** kullan
- Playwright MCP araclari (`mcp__playwright__*`) mevcut ve kullanima hazir

## Ne Zaman Playwright
- AI agent ile web scraping / otomasyon
- Frontend testi ve assertion'lar
- CI/CD pipeline testleri
- Coklu tarayici destegi (Chrome, Firefox, WebKit)
